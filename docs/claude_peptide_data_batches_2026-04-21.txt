General instructions for every batch

Return only valid JSON. No markdown. No prose before or after the JSON.

You are helping populate a peptide reference site for biologists and physicians. Be conservative, literal, and citation-driven.

Do not infer healthy-human enhancement from diseased-population trials.
Do not infer clinical efficacy from cell, animal, or review evidence.
Do not invent missing fields. Use null, [], or "" where appropriate.
Prefer primary sources first:
1. FDA / EMA / regulator pages
2. drug labels / prescribing information
3. peer-reviewed human trials
4. registered clinical trials
5. peer-reviewed animal studies
6. peer-reviewed cell studies
7. reviews only when they add mechanism framing or source discovery

Citation handling rules
- Every factual claim should map to one or more citation IDs.
- Every citation object must be unique within the batch.
- Include accessedAt as YYYY-MM-DD using the date you retrieved the source.
- Use DOI / PMID / official URL when available in notes or title context, but always include url.
- If a source is a review, mark quality as "secondary".
- If a source is a regulator page or label, mark quality as "regulatory".
- If a source is a vendor page, mark quality as "vendor".
- If a source is a primary peer-reviewed paper, mark quality as "primary".

Field guidance
- regulatoryStatus must be one of: fda_approved, non_us_approved, investigational, research_only, not_approved, unknown
- species should be one of: human, animal, mixed
- context should be one of: fda_label, regulatory, clinical_trial, observational, preclinical, cell, review, anecdotal_common_use, vendor
- symbols can use existing site values: Rx, H, A, C, R, V, ?, N
- mechanismSummary should be 1-2 sentences max
- mechanismDetail should be compact but reference-grade and trace actual pathway directionality
- cascades should prefer real nodes over vague summaries
- safetyDetail should distinguish absence of evidence from evidence of absence
- dosing.quick must include context like "FDA-approved obesity dose..." or "preclinical rodent dosing only..."
