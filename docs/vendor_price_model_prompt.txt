Peptocopeia vendor, price, COA, and satisfaction data prompt
Last updated: 2026-04-21

Goal
Compile vendor/manufacturer/price/lab-result rows for rapid import into Peptocopeia. Use public pages only unless the user explicitly provides logged-in exports. Keep all citations and snapshots.

Primary sources to inspect
- Finnrick public pages: https://www.finnrick.com/, /products, /products/{peptide}, vendor/product detail pages, methodology, certificate verification pages.
- PeptideTracker public pages: https://www.peptidetracker.com/dashboard, /peptidedex, /vendors, /peptide/{slug}, /vendors/{slug}. The dashboard is a client-rendered app; inspect bundled JS/network calls and public Supabase REST responses only where access is allowed.
- Vendor product pages and COA pages linked by Finnrick, PeptideTracker, Peptide Partners, or user-provided sources.

Rules
1. Do not treat vendor marketing as efficacy or safety evidence.
2. Store prices as observations with date retrieved, currency, quantity, concentration if stated, and source URL.
3. Store Finnrick-style lab/rating data separately from vendor claims.
4. Preserve COA URLs, lab names, batch IDs, test dates, purity, quantity/assay, sterility, endotoxin, heavy metals, and any fail/flag language exactly.
5. Satisfaction/review data must be labeled as user review, sentiment, rating, complaint, or anecdotal. Never combine it with clinical evidence.
6. If a site blocks scraping, state that and output the URLs/manual fields needed.
7. Capture a source_snapshot object for every row: page title, source URL, date retrieved, visible source notes, and whether data was public, logged-in export, or manually copied.

Validation before returning
- Every vendor_observation must have peptide_id, vendor, product_name, source_platform, source_snapshot.url, and source_snapshot.retrieved_date.
- Every price must include raw text and currency if visible.
- If a numeric field is inferred from visible text, include the visible text in raw and note the inference.
- Do not invent unavailable COA, lab, price, or review data.
